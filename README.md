# Ontology-for-Blue-Amazon

This repository contains the codes and outputs produced for the Master's project [
Ontology extraction and evaluation for the Blue Amazon: an experiment with large language models](https://teses.usp.br/teses/disponiveis/45/45134/tde-24112025-190040/pt-br.html).
It also contains the questionnaire templates used in the experts evaluation of the generated ontologies.

The codes were based on the work described on the publication [Towards Ontology Construction with Language Models](https://www.informatik.uni-leipzig.de/kr/research/papers/2023/onto_llm.pdf).
